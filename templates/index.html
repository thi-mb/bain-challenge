Predict price of milk

This application exposes an endpoint and allows the use of a machine learning model to predict the price of milk based on a few key indicators. It can be used with a GET request, by submitting variables via the url at http://localhost:8000/get_predict/ or via POST request by submitting a JSON using your favorite requests manager to http://localhost:8000/post_predict/. GET requests are limited to one prediction, and are not recommended but are available. The output is always a JSON with the variables and their associated prediction.


Here is an example of a GET request


The POST endpoint requires that all data be merged into a single JSON. Here is an example of a POST request using the python library requests:

import requests
import pandas as pd
import json

df = pd.read_csv('my_data.csv')
my_json = json.loads(df.to_json())
response = requests.post(url='http://localhost:8000/post_request/, json=my_json)
response_json = response.json()
response_df = pd.DataFrame(response_json)
response_df.to_csv('my_predictions.csv')
